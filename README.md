# HomeSure Management

A modern Property Management and Home Warranty Management platform built using React, TypeScript, Vite and TanStack Router.

## Features

- Multi-role Authentication
- Property Management
- Tenant Management
- Contractor Management
- Service Requests
- Warranty Tracking
- Analytics Dashboard
- Notifications

## Tech Stack

- React
- TypeScript
- Vite
- TanStack Router
- Tailwind CSS
.
